# 📐 GarvinLabs: Design System (The Partner)

This system is designed to feel approachable, human-centric, and outcome-focused, while maintaining technical authority through clean typography.

## 1. Color Palette
*   **Primary (Brand Green):** `#10B981` (Matches the logo - signifies growth and "Go" state).
*   **Surface (Warm Neutral):** `#F9FAFB` (Soft off-white to feel approachable).
*   **Ink (Dark Charcoal):** `#1F2937` (Easier on the eyes than pure black).
*   **Accent (Soft Sand):** `#F3F4F6` (For secondary backgrounds and section breaks).
*   **Muted (Slate):** `#6B7280` (For helper text and secondary labels).

## 2. Typography
*   **Heading:** *Figtree* (Semi-bold) - Clean, professional, and trustworthy.
*   **Body:** *Outfit* - A slightly rounder, friendlier sans-serif that feels modern and approachable.
*   **Actionable:** *JetBrains Mono* (Small, for status labels only) - Retains the "Architect" DNA.

## 3. UI Tokens
*   **Corner Radius:** `16px` (Softer and more approachable than sharp industrial corners).
*   **Borders:** `1.5px solid var(--accent-sand)` - Subtle definition.
*   **Shadows:** Soft, diffused shadows to create depth without feeling "heavy."
*   **Grid:** A very subtle dot grid pattern (instead of sharp lines) for background sections.

## 4. Key Components
*   **The "Meet Garvin" Card:** A signature component featuring a high-quality photo and direct "Message Me" link.
*   **The Outcome Badge:** Displays "Reclaimed: 40hrs/wk" style metrics.
*   **The Triage Logic Flow:** A friendly, high-level technical diagram using soft colors.

## 5. Visual Language
*   **Imagery:** Warm-toned lifestyle photography + clean, abstract AI diagrams.
*   **Tone:** Helpful, expert, and results-oriented.
*   **Motion:** Fluid, ease-in-out transitions to feel "Natural" and "Seamless."
